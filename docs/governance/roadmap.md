# Roadmap

## Current Status

CIMP is under active development as an open framework.

- Concepts are stable (Canon v1.0)
- Practices evolve based on real usage
- Tooling is optional

## Near Term

- Complete documentation website
- Additional practice examples
- Community feedback integration

## Long Term

- Continued refinement based on real-world usage
- Expanded examples and case studies
- Community contributions

## How Roadmap Evolves

The roadmap reflects:

- Community needs
- Real-world application feedback
- Governance decisions

For specific proposals, see the [governance process](../docs/GOVERNANCE.md).
