# cimpai.com

## Purpose

This repository contains the source for **[cimpai.com](https://cimpai.com/)** — the official website and documentation hub for CIMP.

The site is:

- documentation-first
- markdown-driven
- built for long-term maintainability
- hosted on GitHub Pages

It aggregates and presents materials from:

- **[cimp](https://github.com/cimpai/cimp)** — core concepts, philosophy, and artifacts
- **[cimp-practices](https://github.com/cimpai/cimp-practices)** — applied patterns, playbooks, and case studies

## What this repository is

- A documentation delivery layer
- A canonical public entry point to CIMP

## What this repository is not

- A product implementation
- A backend or service
- A marketing landing page
