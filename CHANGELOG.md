# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-22

### Added
- Initial release of cimpai.com documentation site
- MkDocs configuration with Material theme
- Custom CIMP branding (white header, neutral colors, custom logo/favicon)
- In-memory markdown preprocessor plugin for automatic list formatting
- GitHub Actions workflow for automated deployment
- Git submodules integration for `cimp` and `cimp-practices` content
- Documentation sections: Home, Docs, Practices, Examples, Governance, Setup
- Local development setup guide
- Custom domain configuration guide
- Git submodules setup guide
- Change planning documentation

### Technical
- Python package setup (`setup.py`) for MkDocs plugin
- Makefile for local development convenience
- Custom CSS for header styling
- HTML template overrides for favicon support
